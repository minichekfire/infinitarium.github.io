Infinitarium

Description

Infinitarium is a sandbox game developed in Unity 3D. The game has no specific goal or objective. The player can explore, create, and experiment with current game features.

Features

  • Advanced NPCs with melee weapons
  • 8 playable maps
  • Wave survival gamemode
  • Cool Melee system
  • Modding support
