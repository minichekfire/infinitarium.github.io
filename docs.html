Function CreateObject(path, name, x, y, z)

Instantiates a prefab from Resources.

  • path: Resource path (e.g. "Prefabs/Cube")
  • name: Name of the new object
  • x, y, z: World position

Returns: boolean (false if prefab not found)

CreateObject("Player", "da player", 0, 1, 0)

Функция CreateObject(путь, имя, x, y, z)

Создаёт объект из префаба Resources.

  • путь: Путь к префабу
  • имя: Имя объекта
  • x, y, z: Позиция в мире

Возвращает: boolean (false, если префаб не найден)

CreateObject("Player", "da player", 0, 1, 0)

Function GetComponent(objName, componentName)

Checks if a GameObject has a component.

  • objName: Name of the GameObject
  • componentName: Name of the component (e.g. "Rigidbody")

Returns: boolean (true if component exists)

if GetComponent("Player", "Rigidbody") then
  printLog("Player has Rigidbody")
end

Функция GetComponent(имяОбъекта, имяКомпонента)

Проверяет наличие компонента на объекте.

  • имяОбъекта: Имя объекта
  • имяКомпонента: Имя компонента (например, "Rigidbody")

Возвращает: boolean (true если компонент существует)

if GetComponent("Игрок", "Rigidbody") then
  printLog("Игрок имеет Rigidbody")
end

  • health, maxHealth, godMode
  • speed, jumpForce, gravity, noclipSpeed
  • maxStamina, currentStamina

player.godMode = true
printLog("Speed:", player.speed)

Таблица player

Глобальная таблица с параметрами игрока. Чтение/запись:

  • maxStamina, currentStamina
