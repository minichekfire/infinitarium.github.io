Infinitarium: Close Quarters

ALPHA

Description

Infinitarium: Close Quarters is a melee combat game focused on close-range fighting mechanics.

Expect a lot of bugs and changes, the game is in alpha.

Features

  • Melee combat system with blocking and parrying
  • Multiple weapons including chopping and crushing weapons
  • Multiplayer with server list
  • Day & night cycle affecting gameplay
  • Work in progress - new features added regularly
